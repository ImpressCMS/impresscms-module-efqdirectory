Manage categories

For the selected directory you can edit its categories in this section.


Below the 'Categories Configuration' header, you will find a table with the categories that have been created for this directory. The title of the category is shown as well as its 'Active status'.


Edit categories

You can edit a category by clicking on its title in this table. The Edit category section will show for this category.


Create new category

To add a new category to the directory you can use the 'Create a new Category' form on the bottom of the page. This form has a number of fields you will need to fill in in order to create the category.

Automatically import default fields from parent category?

If you would like to automatically import the type of data fields from the parent category, you should check this option. When the category selected in the Select parent category form field has any datatypes stored already and you have set one or more of these types to be used in the category's subcategories these datatypes will automatically be copied into the new category.
