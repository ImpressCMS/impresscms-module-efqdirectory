Installation

The EFQ Directory module can be installed like any other module in XOOPS, using the Module administration section in the Administration menu.


CSS files

Mostly this module uses CSS classes that are already in the default XOOPS theme. However, some new classes had to be added for this module. A CSS file has been added to the module directory. If you use a theme that includes the <{$xoops_module_header}> tag or if you use XOOPS version 2.2.x you do not need to change anything.


In XOOPS versions 2.0.x if you do not have a theme that includes the <{$xoops_module_header}> tag you should either add this tag to the theme afterall, or copy the CSS code from theme.css to your theme's CSS file.
