Create subcategory

You can create subcategories not only from the 'Manage categories' section (by selecting a parent category) but also from the 'Edit category' section.


At the bottom of the 'Edit category' page the form 'Create new subcategory' is shown. The form is almost the same is in the 'Manage categories' section, the difference being that by default the 'Automatically import default fields from parent category?' option is set to active and the the 'Parent category' selection box is set to the category that is being edited.
