Add data field type

Using the 'Add data field type form' the admin user can add data field types to a category. Data field types are basically the fields that are linked to a category.


For each data field type you enter the following fields:

               

Form field

Description

Mandatory

Default

Data field type title

Title of the data field type.

Yes


Select icon

Select an icon for the data field type. This will be shown in the submit listing form.

No


Field type name

Select the field type that is associated with this data field type. Currently you can chose from 'Address', 'DHTML text area', 'Select box', 'Text area', 'Textbox default', 'Yes/No'.

Yes


Options

This field needs to be used when field type name 'Select box' is selected. The options for the select box will be added using the options field.

No


Set this data field type active?

Set the data field active or not. Check this box if you want to activate the data field type.

No

Off

Set this data field type as default for child categories?

Set the data field to be copied into child categories by default. Check this box if you want to set the data field type as default for child categories.

No

Off

Sequence

Set the sequence of this data field type. The sequence is used upon creating the 'submit listing' and 'edit listing' form.

No

0

Section

Determin in which section the data field type should be shown. By default there are 3 sections. More about sections

No

0


Example

For a library directory you can add the fields 'ISBN number', 'Price', 'Authors', 'Rating' to the subcategory 'Thrillers' in the category 'Books'.


Data field type 'ISBN number', 'Price', 'Authors' would have to be created as text fields.

Data field type 'Rating' would have to be created as a select box field, where options need to be added to the data field type.


Sections

In EFQ Directory you can chose the way you want to show the listing and more particurlarly its data. In the efqdirectory_listing.html template file there are 3 sections of data in the default installation. You can easily add nother section or change the way listings are presented. The code in the module allows you to add an virtually unlimited number of sections to the template without changing the code in the php files.
