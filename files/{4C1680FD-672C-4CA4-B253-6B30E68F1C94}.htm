Delete category

You can delete a category from the 'Edit category' page. Just below the 'Edit category form' you will find the 'Delete category' link. Upon clicking the link, you will be asked to confirm if the category should be deleted. If you confirm by clicking the 'Delete' button the category will be deleted.


The listings stored under a deleted category will remain to exists, because it may as well been linked to other categories as well.


Future planning: View listings that are not linked to any category

In this release it is not yet possible for the admin user to view listings that are not linked to any category. This feature is planned for a future release.
