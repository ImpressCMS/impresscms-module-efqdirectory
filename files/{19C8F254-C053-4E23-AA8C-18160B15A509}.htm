Manage directories

If the module has one or more directories you can edit them using the 'Manage directories' feature. A table will show you the directories that have been created. Clicking on a directory name will get you to the Edit directory page. In the last column you will be able to select the link to the Manage categories page where you can manage categories for the selected directory.
