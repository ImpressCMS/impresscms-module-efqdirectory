Submit new listing

To submit a new listing a registered user can use the "submit form" from within a directory. When the user is browsing within a directory a link to the 'submit listing' page is visible in the top right corner.


First the title field and at least one category need to be entered by the user and afterwards the button 'Continue' need to be clicked to save the listing. The user will be redirected to the "edit listing page" for the saved listing.


Edit new listing

In the "edit listing page" the user can fill in the rest of the fields for the categories that are linked to this listing. Fields which are always present are the 'description' field and the 'upload picture' field.


Submit for approval

The 'Submit for approval' button is used for completing the listing and to notift the admin that a new listing is ready to be approved (or rejected).
